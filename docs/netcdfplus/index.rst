.. _netCDF:

netCDF+ package
===============


Overview
--------

* Cool format.
* Allows independent frame storage positions, velocities, including lots of additional objects necessary.
* Easy export to :py:mod:`mdtraj` native objects possible so export to all sorts of file formats possible.
* Most objects are stored using human readable JSON String of Python objects.
